# Shared UI Components

Reusable UI components shared across JCW suite applications.

## Planned Components
- `AppDock` — Navigation dock with service buttons
- `Header` — Consistent top bar with branding and user menu
- `DataTable` — Sortable, filterable data tables
- `StatusBadge` — Status indicators for approvals, entries, etc.
- `styles/` — Shared CSS variables, themes, and base styles
