# Cost Coding: CSI MasterFormat for JCW Construction

**Research Date:** February 2026  
**Prepared for:** JCW Construction

---

## Executive Summary

The CSI MasterFormat is the industry-standard coding system used by construction companies to organize project costs consistently. Using standardized cost codes enables accurate job costing, seamless data sharing with QuickBooks and other systems, and better historical data for future bidding. JCW Construction should adopt a tailored subset of MasterFormat codes that aligns with their project types and integrates with their Labor Timekeeper system.

---

## Key Findings

### 1. About CSI MasterFormat

The Construction Specifications Institute (CSI) MasterFormat is the industry standard for organizing construction cost data. The current version (2024 edition) uses a 16-division structure for many construction activities.

### 2. Core Divisions for Most Construction Companies

| Division | Description |
|----------|-------------|
| 01 | General Requirements - Project management, supervision |
| 02 | Existing Conditions - Demolition, site prep |
| 03 | Concrete - Foundations, slabs, formwork |
| 04 | Masonry - Brick, block, stone work |
| 05 | Metals - Structural steel, metal framing |
| 06 | Wood, Plastics, Composites - Framing, carpentry |
| 07 | Thermal and Moisture Protection - Roofing, insulation |
| 08 | Openings - Doors, windows, glazing |
| 09 | Finishes - Drywall, painting, flooring |
| 10 | Specialties - Cabinetry, equipment |
| 21 | Fire Suppression - Sprinklers, fire protection |
| 22 | Plumbing - Piping, fixtures |
| 23 | HVAC - Heating, cooling, ventilation |
| 26 | Electrical - Wiring, lighting, panels |
| 31 | Earthwork - Excavation, grading |

### 3. Recommended Cost Code Structure for JCW

**Labor Cost Codes:**
- 01-1000: Project Management (superintendent, PM)
- 01-2000: Supervision (foreman, lead)
- 03-1000: Concrete Labor
- 03-2000: Formwork Labor
- 04-1000: Masonry Labor
- 05-1000: Steel Labor
- 06-1000: Carpentry Labor
- 09-1000: Drywall Labor
- 09-2000: Painting Labor
- 31-1000: Excavation Labor

**Material Cost Codes:**
- 03-3100: Concrete Materials
- 03-3200: Formwork Materials
- 04-1000: Masonry Materials
- 05-1000: Steel Materials
- 06-1000: Lumber and Materials

**Equipment Cost Codes:**
- 01-3000: Small Tools
- 01-4000: Equipment Rental
- 01-5000: Heavy Equipment

### 4. Implementation Best Practices

- **Start Simple**: Use 20-30 core codes, expand as needed
- **Match QuickBooks Items**: Align codes with QuickBooks service items
- **Train All Users**: Ensure field and office staff understand codes
- **Use Consistent Naming**: Standard abbreviations and descriptions
- **Document Everything**: Maintain a code dictionary reference

---

## Recommendations for JCW Construction

1. **Adopt 16-Division Structure**: Use MasterFormat as the foundation
2. **Create JCW-Specific Subset**: 30-50 codes covering 80% of work
3. **Map to Labor Timekeeper**: Each time entry requires a cost code
4. **Sync with QuickBooks**: Match codes to QuickBooks products/services
5. **Standardize Reports**: Use cost codes for all financial reporting
6. **Review Quarterly**: Update codes as project types evolve
